# 迭代器

迭代器周边有几个基础概念，分别是:

  • 可迭代对象
  • 迭代器对象
  • 迭代结果

# 可迭代对象

具有专用迭代器方法（以 Symbol.iterator 命名），且该方法返回迭代器对象的对象

# 迭代器对象

具有 next 方法，且该方法返回迭代结果对象的对象

# 迭代结果对象

具有 value 和 done 属性的对象
